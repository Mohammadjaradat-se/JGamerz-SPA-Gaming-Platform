# JGamerz - SPA Gaming Platform

JGamerz is a gaming platform developed as part of a Graduation Project.

The frontend was built using Vanilla JavaScript, HTML, and CSS without using frameworks such as React or Angular, following the Single Page Application (SPA) architecture pattern.

My primary contribution focused on the complete frontend development of the platform, including UI implementation, SPA architecture, routing, gaming feed, community features, chat interfaces, user profiles, marketplace pages, gaming news pages, and integration with backend services.

---

## Project Overview

JGamerz is a gaming platform designed to provide gamers with a centralized environment where they can explore games, interact with the community, stay updated with gaming news, browse marketplace content, and manage their profiles through a modern and responsive user experience.
